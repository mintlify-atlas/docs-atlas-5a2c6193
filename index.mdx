Dashboard Icons is a community-maintained collection of high-quality icons for services, applications, and tools — designed specifically for self-hosted dashboards like Homarr, Homepage, Dashy, and Heimdall.

Every icon is available in multiple formats and delivered globally via CDN, so you can drop them into any dashboard with a single URL.

## Get started

<Steps>
  <Step title="Find your icon">
    Browse or search the full collection at [dashboardicons.com](https://dashboardicons.com) to find the icon you need.
  </Step>
  <Step title="Choose your format">
    Pick between SVG (vector), PNG (512px), or WebP (512px) — all available from the same CDN path.
  </Step>
  <Step title="Copy the CDN link">
    Use the jsDelivr CDN URL pattern to reference icons directly in your dashboard configuration:
    ```
    https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/plex.svg
    ```
  </Step>
  <Step title="Add to your dashboard">
    Paste the URL into your dashboard's icon field. No downloads required.
  </Step>
</Steps>
