---
title: "Quick Start"
description: "Start using Dashboard Icons in your dashboard in minutes"
---

Dashboard Icons requires no installation, account, or API key. Icons are served directly over CDN — you just need to know the URL pattern.

<Steps>
  <Step title="Find an icon">
    Browse the collection to find the icon you need.

    <Note>
      The easiest way is to search at [dashboardicons.com](https://dashboardicons.com). You can preview icons in light and dark themes and copy CDN links directly from the site.
    </Note>

    You can also browse the repository directly on [GitHub](https://github.com/homarr-labs/dashboard-icons) — icons are organized by format in the `svg/`, `png/`, and `webp/` directories.

    <Tip>
      Icons use **kebab-case** naming. Spaces and special characters become hyphens and everything is lowercase. For example:
      - "Nextcloud Calendar" → `nextcloud-calendar`
      - "Pi-hole" → `pi-hole`
      - "AdGuard Home" → `adguard-home`
    </Tip>
  </Step>

  <Step title="Build the CDN URL">
    All icons are available via jsDelivr using this pattern:

    ```
    https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/{format}/{icon-name}.{format}
    ```

    Replace `{format}` with `svg`, `png`, or `webp`, and `{icon-name}` with the kebab-case icon name.

    **Examples:**

    <CodeGroup>
      ```bash SVG
      https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/plex.svg
      ```

      ```bash PNG
      https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/png/plex.png
      ```

      ```bash WebP
      https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/webp/plex.webp
      ```
    </CodeGroup>
  </Step>

  <Step title="Use the icon">
    Reference the CDN URL wherever your dashboard or application accepts an image URL.

    **In HTML:**

    ```html
    <img src="https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/plex.svg" alt="Plex">
    ```

    **Download with curl or wget:**

    ```bash
    # Download with curl
    curl -O https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/nextcloud.svg

    # Download with wget
    wget https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/nextcloud.svg
    ```

    Most dashboard applications (Homarr, Homepage, Dashy, Heimdall) have an icon URL field in their service or app settings — paste the CDN URL directly there.
  </Step>

  <Step title="Use light and dark variants">
    Some icons ship in theme-specific variants. Use the `-light` or `-dark` suffix before the file extension to select the right one for your theme.

    | Variant | Suffix | Best for |
    |---------|--------|----------|
    | Light | `-light` | Dark backgrounds |
    | Dark | `-dark` | Light backgrounds |

    **Examples:**

    <CodeGroup>
      ```bash Light variant (for dark backgrounds)
      https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/github-light.svg
      ```

      ```bash Dark variant (for light backgrounds)
      https://cdn.jsdelivr.net/gh/homarr-labs/dashboard-icons/svg/github-dark.svg
      ```
    </CodeGroup>

    Icons without a `-light` or `-dark` suffix are designed to work on both backgrounds.
  </Step>
</Steps>
